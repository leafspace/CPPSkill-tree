职位信息

岗位职责：

1. 根据软件概要设计说明书，进行产品生产检测系统的模块化设计； 

2. 根据软件详细设计说明书进行代码设计、编写、调试； 

3. 负责软件单元测试； 

4. 负责软件相关设计缺陷的修改闭环； 

5. 负责软件发布及发布前技术文件的准备。

岗位要求：

1. 熟悉C++、MFC开发； 

2. 熟悉串口通讯； 

3. 熟悉html、js开发；

4. 计算机相关专业，本科及以上学历，1年或以上软件开发经历。

职能类别： 系统工程师 软件工程师

关键字： 工装开发 C++

公司信息

  浙江威星智能仪表股份有限公司成立于2005年，于2017年2月在深圳证券交易所中小板挂牌上市，股票简称：威星智能，股票代码：002849。
  威星智能自成立以来，专业致力于为城市公用事业提供智能计量终端及燃气管理系统平台的研发、生产和销售，经过多年的发展，已经成为国内领先的城市燃气行业智能燃气表制造商和系统供应商。
  威星智能目前拥有3大产品开发体系，智能产品平台、计量产品平台以及软件产品平台，向城市燃气运营商提供用气计量智能终端、运营管理（抄表、结算、收费、设备管理、用气分析等）和数据及其他服务。
  威星智能作为国家高新技术企业，公司的研发中心经浙江省科技厅认定为省级科技研发中心，经省经信委认定为省级企业技术中心，经省发改委认定为省工程研究中心，并于2014年8月被认定为浙江省重点企业研究院。
  威星智能的产品获得了国家计量器具制造许可证、计量器具型式批准证书、防爆许可证等认证。公司的多项成果被列入浙江省、杭州市科研及产业化项目，并拥有一百多项国家专利和软件著作权。公司产品被认定为杭州市名牌产品。公司先后获得了AAA级信用企业、省级专利示范企业等称号，通过ISO9001、14000、18001体系认证，具备优质的产业基础和良好的发展前景。
  威星智能拥有朝气、勤勉、务实的员工队伍和国内领先的生产检测设备、先进的智能燃气表整机生产线，能保证产品的高品质及迅速交付；杰出的研发设计团队可应不同客户要求而量身设计大型配套电子系统，提供多元化的产品和服务；严格的品质管理，在每道工序都设置了严格的质量监控，反复筛选，层层把关，确保了产品的可靠性与安全性。
  威星智造，无限可能！
